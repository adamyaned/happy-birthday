# 🎂 Birthday Surprise Card

An interactive birthday greeting card built with HTML, CSS, and JavaScript, featuring multiple screens, animations, and festive effects.

## ✨ Features

* Welcome screen with a start button
* Animated terminal-style loading sequence
* Interactive birthday cake with candles
* Click-to-blow-out candle interaction
* Confetti celebration effect
* Floating celebration balloons
* Optional background music
* Responsive design for desktop and mobile devices
* Support for `prefers-reduced-motion` accessibility settings

## 📂 Project Structure

```text
.
├── index.html
├── happy-birthday.mp3
└── README.md
```

## 🚀 Getting Started

### Option 1: Open Directly

1. Download or clone the repository.
2. Place the `happy-birthday.mp3` file in the project root directory.
3. Open `index.html` in your browser.

### Option 2: Run a Local Server

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## 🎮 How It Works

### 1. Welcome Screen

The user is greeted with a simple introduction and a **Start** button.

### 2. Terminal Animation

A playful terminal-style loading sequence simulates the preparation of the birthday surprise:

* Finding the birthday person
* Loading smiles
* Gathering good vibes
* Preparing the cake and candles
* Packaging everything into a greeting card

### 3. Birthday Card

The final screen displays an interactive birthday cake.

Users can:

* Click candles to blow them out
* Reveal a birthday message
* Trigger a confetti celebration
* Watch balloons float across the screen
* Relight the candles and replay the experience
